Projects

Twitter Sentiment Analysis

This project performs sentiment analysis on Twitter data using a Logistic Regression model to classify tweets as positive or negative. It involves preprocessing text by converting it to lowercase, removing URLs, punctuation, and numbers, tokenizing, removing stopwords, and lemmatizing. The text is then vectorized using TF-IDF for machine learning. The Logistic Regression model is trained on this data to identify sentiment patterns. Model performance is evaluated with accuracy and classification reports. The project also includes functionality to predict sentiment for new text inputs, making it useful for real-time analysis. The dataset used is Sentiment140, containing 1.6 million labeled tweets.

TCP/IP Client-Server Interaction

This project involves creating a simple TCP/IP server and client program. The server listens on port 8080, reads hexadecimal strings sent by clients, converts them to integers, and prints the results along with the current time. The client connects to the server, reads hexadecimal strings from a file, and sends them to the server, printing the current time and data sent. Both programs include robust error handling to ensure reliable operation, covering socket creation, binding, listening, connection acceptance, data reading, address conversion, and file operations. This project provides practical experience in TCP/IP networking, socket programming, and error handling in C.
